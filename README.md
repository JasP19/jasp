# jasp

This project is a personal blog site for coding, electronics and gaming-related posts. It is based on the [Eleventy](https://www.11ty.io/) static site generator as well as [7ty.tech](https://7ty.tech). It features a Vue.js powered simple search as well as pagination, animations and much more.

## Quickstart

- `Clone` or `download` the repo
- Run `npm install` to install dependencies
- Run `npm start` to start the development server.
- Point your browser to `localhost:8080`
- Run `npm run build` to build the project for deployment.
- Run `firebase serve --only hosting` to test locally
- Run `firebase deploy` to deploy
